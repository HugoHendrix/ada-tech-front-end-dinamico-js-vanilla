Exercício 03 - Alteração de Temas

Este é um conjunto de exercícios desenvolvidos como parte do Programa Vem Ser Tech, uma parceria entre a Ada Tech e o iFood.

Voltar para página principal

O Exercício 03 envolve a manipulação dinâmica de temas em uma página da web. Através do JavaScript Vanilla, você pode alterar o visual da página entre um tema claro e um tema escuro.

Este exercício é uma ótima oportunidade para praticar a manipulação de estilos e temas em tempo real usando JavaScript puro.
